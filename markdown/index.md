# Markdown 标记符用法全解析

## 一、标题（Heading）

标题用于划分文档结构，Markdown 支持六级标题，通过 `#` 符号加空格实现，层级由 `#` 数量决定（`#` 越少，层级越高）。

### 语法示例：

```markdown
# 一级标题（最大层级）
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题（最小层级）
```

### 显示效果：

# 一级标题（最大层级）
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题（最小层级）

## 二、文本格式（Text Formatting）

### 1. 加粗（Bold）

**作用**：突出关键信息  
**语法**：用 `**` 包裹文本  
**示例**：  

```markdown
这是 **加粗文本**，用于强调重点。
```  

**效果**：这是 **加粗文本**，用于强调重点。

### 2. 斜体（Italic）

**作用**：表示引用、书名或强调语气  
**语法**：用 `*` 或 `_` 包裹文本  
**示例**：  

```markdown
*斜体文本* 常用于引用，_也可用下划线_ 实现。
```  

**效果**：*斜体文本* 常用于引用，_也可用下划线_ 实现。

### 3. 加粗斜体（Bold + Italic）

**作用**：叠加加粗与斜体效果  
**语法**：用 `***` 包裹文本  
**示例**：  

```markdown
***同时加粗和斜体*** 的文本。
```  

**效果**：***同时加粗和斜体*** 的文本。

### 4. 删除线（Strikethrough）

**作用**：标记已删除或错误内容  
**语法**：用 `~~` 包裹文本  
**示例**：  

```markdown
这是 ~~删除线文本~~，表示内容无效。
```  

**效果**：这是 ~~删除线文本~~，表示内容无效。

## 三、列表（Lists）

### 1. 无序列表（Unordered List）

**作用**：展示并列关系的条目  
**语法**：用 `-`、`+` 或 `*` 开头（推荐用 `-`），后跟空格  
**示例**：  

```markdown
- 苹果  
- 香蕉  
- 橙子  
```  

**效果**：  

- 苹果  
- 香蕉  
- 橙子  

### 2. 有序列表（Ordered List）

**作用**：表示顺序或步骤  
**语法**：用数字加 `.` 开头，后跟空格（数字可重复，Markdown 会自动渲染为连续序号）  
**示例**：  

```markdown
1. 第一步：打开文档  
2. 第二步：输入内容  
3. 第三步：保存文件  
```  

**效果**：  

1. 第一步：打开文档  
2. 第二步：输入内容  
3. 第三步：保存文件  

### 3. 嵌套列表（Nested List）

**作用**：表示层级关系  
**语法**：在子项前缩进 2 个空格或 1 个制表符  
**示例**：  

```markdown
- 主项 1  
  - 子项 1.1  
  - 子项 1.2  
- 主项 2  
  1. 子项 2.1  
  2. 子项 2.2  
```  

**效果**：  

- 主项 1  
  - 子项 1.1  
  - 子项 1.2  
- 主项 2  
  1. 子项 2.1  
  2. 子项 2.2  

## 四、引用（Blockquote）

**作用**：引用他人观点或文献  
**语法**：用 `>` 开头，后跟空格（可嵌套多层引用，每层增加一个 `>`）  
**示例**：  

```markdown
> 这是一级引用。  
> > 这是嵌套的二级引用。  
```  

**效果**：  
> 这是一级引用。  
> > 这是嵌套的二级引用。  

## 五、代码（Code）

### 1. 行内代码（Inline Code）

**作用**：标记少量代码或术语  
**语法**：用 `` ` `` 包裹文本  
**示例**：  

```markdown
使用 `print("Hello World")` 输出问候语。  
```  

**效果**：使用 `print("Hello World")` 输出问候语。  

### 2. 代码块（Code Block）

**作用**：展示大段代码，支持语法高亮  
**语法**：用 ``` 包裹代码（可在开头指定编程语言，如 `python` `javascript`）  
**示例**：  

```markdown
```python
def greet():  
    print("Hello World")  
greet()  
```  
```  

**效果**（Python 语法高亮）：  

```python
def greet():  
    print("Hello World")  
greet()  
```  

## 六、链接（Links）

### 1. 超链接（Hyperlink）

**作用**：跳转至网页或文档  
**语法**：`[链接文本](URL)`  
**示例**：  

```markdown
访问 [百度](https://www.baidu.com)  
```  

**效果**：访问 [百度](https://www.baidu.com)  

### 2. 引用式链接（Reference Link）

**作用**：分离链接内容与文本，适合长链接  
**语法**：  

```markdown
[链接文本][标签]  
[标签]: URL  
```  

**示例**：  

```markdown
查看详细说明 [点击这里][doc]  
[doc]: https://example.com/document  
```  

**效果**：查看详细说明 [点击这里][doc]  

## 七、图片（Images）

**作用**：插入图片  
**语法**：`![替代文本](图片URL "可选标题")`  
**示例**：  

```markdown
![Markdown 图标](https://example.com/markdown-icon.png "Markdown Logo")  
```  

**效果**（若图片存在）：![Markdown 图标](https://example.com/markdown-icon.png "Markdown Logo")  

## 八、表格（Tables）

**作用**：结构化展示数据  
**语法**：  

- 用 `|` 分隔列，用 `-` 分隔表头与内容  
- 表头行下方需添加分隔线（`---` 表示左对齐，`:---` 右对齐，`:-:` 居中）  

**示例**：  

```markdown
| 姓名 | 年龄 | 职业       |  
|------|------|------------|  
| 张三 | 25   | 工程师     |  
| 李四 | 30   | 设计师     |  
```  

**效果**：  

| 姓名 | 年龄 | 职业       |  
|------|------|------------|  
| 张三 | 25   | 工程师     |  
| 李四 | 30   | 设计师     |  

## 九、分隔线（Horizontal Rule）

**作用**：分隔文档章节  
**语法**：用 `---`、`***` 或 `___`（需单独占一行，前后空行）  
**示例**：  

```markdown
内容结束  

---  

新章节开始  

```  

**效果**：

内容结束  

---  

新章节开始  

## 十、转义字符（Escape Character）

**作用**：显示 Markdown 符号本身  
**语法**：在符号前加 `\`  
**示例**：  

```markdown
输出 `# 不是标题`：\# 不是标题  
```  

**效果**：输出 `# 不是标题`：\# 不是标题  

## 总结：常用标记符速查表

| 功能         | 标记符/语法                          | 示例                          |
|--------------|--------------------------------------|-------------------------------|
| 标题         | `# 一级标题` 至 `###### 六级标题`    | `## 二级标题`                 |
| 加粗         | `**文本**`                           | `**重要内容**`                |
| 无序列表     | `- 条目`                             | `- 苹果`                      |
| 代码块       | ```python<br>代码<br>```             | ```python<br>print()<br>```   |
| 链接         | `[文本](URL)`                        | `[百度](https://baidu.com)`    |
| 表格         | `| 列1|列2|\n|---|---|\n|内容|内容|` | 见上文表格示例                |

通过合理使用这些标记符，可高效排版文档、技术笔记、博客等内容，提升可读性与专业性！
